# SoftwareMerger 公开更新库

本仓库仅用于 **SoftwareMerger 版本发布与在线更新**，不包含源码。

- 源码私有仓库：`SoftwareMerger`（不公开）
- 更新清单：`update.json`

## 当前最新版本

| 组件 | 版本 |
|------|------|
| 软件组合制作器（Builder） | **0.1.1** |
| 一键启动模板（Launcher） | **0.1.1** |

## 老客户如何升级

### 方式一：制作器内在线更新（推荐，0.1.1 起）

1. 打开 `SoftwareMergerBuilder.exe`
2. 等待自动检查更新，或点击 **检查更新**
3. 按提示下载并应用更新

### 方式二：手动下载

1. 打开 [Releases](https://github.com/e12games/SoftwareMerger-releases/releases)
2. 下载 `SoftwareMergerBuilder-0.1.1-win64.zip`
3. 解压并覆盖原安装目录

## v0.1.1 更新说明

- 新增 Builder 在线更新功能
- 中文夜色护目界面
- 高级内置组件（三击说明 + 密码进入）
- 导出组合包启动器命名为「一键启动.exe」
- Qt 运行库编译后自动部署
